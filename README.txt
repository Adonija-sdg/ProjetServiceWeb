Cette application API permet à l'entreprise Dock-Dorsal de gérer les saisies journalières de correction apportée par leurs produits pour la posture du dos. Les utilisateurs peuvent s'inscrire, saisir leurs corrections quotidiennes, effectuer des modifications ou des suppressions, consulter leurs données et remonter les saisies sur une période donnée.

Technologies Utilisées
Java
Spring Boot
Spring Security
Spring Data JPA
PostgreSQL (ou toute autre base de données de votre choix)


Configuration
Clonez le repository :https://github.com/Adonija-sdg/ProjetServiceWeb.git
Configurez votre base de données dans le fichier src/main/resources/application.properties.
port utilisé 8082 
URL de la BD: postgresql://localhost:5432/DockDorsal_service
